# @kitmi/data

## Jacaranda Framework Data Access Model

`@kitmi/data` 

## Installation

To install `@kitmi/data`, run the following command:

```bash
bun add @kitmi/data
```

Or if you're using npm:

```bash
npm install @kitmi/data
```
